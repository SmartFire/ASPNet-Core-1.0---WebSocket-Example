# Welcome to ASP.NET Core

## This application consists of:

*   Sample pages using ASP.NET Core MVC
*   Theming using [Bootstrap](http://go.microsoft.com/fwlink/?LinkID=398939)
*   Websocket Example


## Run & Deploy

*   dotnet restore
*   dotnet run
*   access http://localhost:5000/Home/About
